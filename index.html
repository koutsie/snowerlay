Privacy Policy

Snowerlay is a Twitch Video Overlay extension that draws a falling snow animation on top of the video player. It runs entirely in your browser.

We do not collect, store, transmit, or process any data. Specifically, Snowerlay:

  • Does not use cookies, local storage, or any form of tracking
  • Does not request your Twitch identity, user ID, or channel information
  • Does not make any network requests to any server, ours or otherwise
  • Does not use analytics, ads, or third-party scripts of any kind
  • Has no backend — there is no server for data to go to

The extension consists solely of static HTML, CSS, and JavaScript files served by Twitch's extension hosting, plus the official Twitch Extension Helper script, which is loaded directly from Twitch (extension-files.twitch.tv) and governed by Twitch's own privacy policy.

Because nothing is collected, there is nothing to delete, export, or opt out of. If this ever changes in a future version, this page will be updated first.
